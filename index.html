This JavaScript-based tool uses a modified Vigenère cipher to encrypt and decrypt messages using a user-provided key. It ensures the output contains only alphanumeric characters (0-9, A-Z, a-z), making it safe for sharing and storage. The encryption runs entirely on the client side without external dependencies. It's ideal for simple secure communication, obfuscation, or learning cryptographic concepts.
